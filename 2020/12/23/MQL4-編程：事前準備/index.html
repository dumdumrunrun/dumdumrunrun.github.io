MQL4 編程：事前準備

Posted on 2020-12-23 Edited on 2020-12-31

在正式開始寫 MQL4 代碼前，我們需要先準備好 MQL4 的開發環境以及了解我們常用的查詢網站有哪些。

開發環境

我們要用到的開發環境 MetaEditor 及 MetaTrader 4 只需到 MetaTrader 4 官網 下載安裝 MetaTrader 4 版即可。

  • 目前 Windows 和 Mac OS 皆可支援（詳細資訊請參見官網）。
  • MetaTrader 4 只是 經紀商 和 交易者 之間的媒介，所以你可以使用 MetaTrader 4 連上任何一家使用它作為報價的經紀商 。

查詢網站

幫我們在開發過程中遇到需要查詢資料、尋找靈感、充實交易知識時，以下是幾個常用網站。

  • MQL4 Book

    MQL4 的基礎知識

  • MQL4 Docs

    MQL4 的參考文件，查詢語法、函式功能

  • MQL4 Articles

    開發者發表文章的地方

  • MQL4 Code Base

    各種免費源代碼，專家顧問、指標、腳本、庫

  • Babypips

    一系列循序漸進的外匯保證金交易的課程
